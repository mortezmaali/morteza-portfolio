Morteza Portfolio Website - Upload Instructions

This is a simple static website.
Files included:
- index.html
- styles.css

Fastest free ways to publish:

Option 1 - GitHub Pages
1. Create a new GitHub repository, for example: morteza-portfolio
2. Upload index.html and styles.css into the root of that repository
3. Go to repository Settings -> Pages
4. Under "Build and deployment", choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save
6. After about 1-3 minutes, GitHub will give you a live website URL

Option 2 - Netlify
1. Go to Netlify
2. Choose "Add new site" -> "Deploy manually"
3. Drag and drop the whole folder contents
4. Netlify will instantly publish the site and give you a URL
